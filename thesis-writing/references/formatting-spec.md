# 绵阳城市学院本科毕业论文（设计）撰写规范

> 来源：《绵阳城市学院本科毕业论文（设计）撰写规范》——实践教学与基地管理处

---

## 1 结构要求

完整的毕业论文（设计）由以下八部分组成，排列顺序为：

1. 封面（学校统一制作）
2. 学术诚信声明和使用授权声明（学校统一模板）
3. 摘要（中文 + 英文，各占一页，中文在前）
4. 目录（三级标题）
5. 正文（含绪论和结论）
6. 致谢（无章节号，≤1000 字）
7. 参考文献（无章节号）
8. 附录（必要时，无章节号）

### 1.1 摘要要求

- 中文摘要约 **300 字**，涵盖研究目的、方法、结果和结论，重点是结果和结论。
- 不使用图、表、化学结构式、非公知公用的符号和术语。
- 摘要正文下空一行顶格打印"关键词"，列出 **3~8 个**关键词。
- 中文关键词之间用"**；**"加一个空格隔开；英文关键词之间用"**,**"加一个空格隔开；最后一个关键词后不加标点。
- 英文摘要另起一页，内容与中文对应，外文一律 Times New Roman 体。

### 1.2 正文结构

- **绪论（引言）**：作为第 1 章，含研究背景与目的、国内外研究现状（文献综述）、研究方法及内容。
- **主体**：各章分述研究内容，结构合理、层次清楚、重点突出。
- **结论**：单独一章，无章节号。准确、完整、精练地阐述创造性工作或新见解及其意义。

## 2 工作量要求（理工类/艺术）

| 项目 | 要求 |
|------|------|

| 参考文献总数 | ≥ **20** 篇 |

| 外文文献 | ≥ **3** 篇 |

| 网上文献 | 不计入规定数量 |

## 4 页面设置

### 4.1 页边距

| 位置 | 尺寸 |
|------|------|
| 上边距 | 3.0 cm |
| 下边距 | 3.0 cm |
| 左边距 | 3.0 cm |
| 右边距 | 2.5 cm |
| 页眉 | 2.0 cm |
| 页脚 | 2.0 cm |

### 4.2 页眉

- 内容：**绵阳城市学院本科生毕业论文**
- 字体：五号华文行楷
- 边框线型：上细下粗

### 4.3 页码

- 正文（绪论起至附录）：五号宋体阿拉伯数字连续编排，页面底端居中。
- 封面：不编页码。
- 摘要和目录：罗马数字（Ⅰ、Ⅱ、Ⅲ），位于页眉右边。

---

## 5 正文格式

### 5.1 基本格式

- 全文无网格，行距固定值 **22 磅**。
- 段前段后不空行（图、表处可适当调整）。

### 5.2 标题样式（理工类/艺术）

| 级别 | 样式 |
|------|------|
| 一级标题 | 另起一页，居中，段前空 1 行，段后空 0.5 行 |
| 二级标题（理工类） | 左起空两字符，段前空 0.25 行，段后空 0.25 行 |
| 三级标题 | 左起空两字符，段前空 0.25 行，段后空 0.25 行 |

- 章、节编号全部顶格排，编号与标题之间空 1 个字的间隙。
- 标题字数一般 ≤15 字，不使用标点符号。
- 标题不能单独置于页面最后一行（禁止孤行）。

### 5.3 层次编号（理工类/艺术）

```
章：  1、2、3……（如"第1章""1 绪论"）
节：  1.1、1.2……
条：  1.1.1、1.1.2……
款/项：（1）、（2）……
```

---

## 6 引用文献标注

- 采用**顺序编码制**：按正文中引用出现的先后顺序连续编码。
- 标注方式：**上标**，阿拉伯数字置于方括号中，放在所引内容最末句右上角。
  - 示例：`……成果[1]`
- 直接提及时用小 4 号字正文排齐：`由文献[8，10~14]可知`
- 不得将引用文献标示置于各级标题处。

---

## 7 表格规范

- 使用**三线表**，与文字齐宽，行高 0.9 厘米。
- 顶线、底线线粗 **1.5 磅**；栏目线线粗 **0.5 磅**。
- 表序按章编排：`表 1-1`、`表 1-2`……
- 表序与表名置于**表上居中**，5 号宋体加粗，数字和字母为 5 号 Times New Roman 加粗。
- 表序与表名之间空一格，表名不使用标点、末尾不加标点。
- 表中用字一般五号字（最小不小于七号）。
- 转页时重复编号后跟"（续）"，如 `表 1-1（续）`，重复表头。
- 附注用小 5 号宋体写在表下方；单条附注写"注："，多条用"注 1："。

---

## 8 公式规范

- 另起一行居中编排，较长公式在等号或运算符后换行。
- 编号用小括号，右对齐顶格，按章编排：`（1-1）`、`（1-2）`……
- 编号前不写"公式"或"式"字样。
- "式中"两字左起顶格，后接符号解释；解释间用"；"隔开。
- 引用时写"见公式（1-1）"或"由公式（1-1）"。

---

## 9 插图规范

- 图号按章编排：`图 1-1`、`图 1-2`……
- 图题置于**图下居中**，5 号宋体加粗，数字和字母为 5 号 Times New Roman 加粗。
- 图名在图号后空一个字的间隙。
- 分图编号用 (a)、(b)、(c)……
- 插图须紧跟文述，先文后图。
- 插图与图题不得拆分到两页。

---

## 10 参考文献格式（GB/T 7714-2005）

### 10.1 类型标识

| 类型 | 标识 |
|------|------|
| 期刊文章 | [J] |
| 普通图书 | [M] |
| 论文集 | [C] |
| 学位论文 | [D] |
| 专利 | [P] |
| 标准 | [S] |
| 报纸文章 | [N] |
| 报告 | [R] |

### 10.2 著录格式

**期刊文章：**
```
[序号] 作者.文献题名[J].刊名,出版年份,卷号(期号):起-止页码.
```

**学术著作：**
```
[序号] 作者.书名[M].版次(首次免注).翻译者.出版地:出版社,出版年:起-止页码.
```

**有 ISBN 号的论文集：**
```
[序号] 作者.题名[A].主编.论文集名[C].出版地:出版社,出版年:起-止页码.
```

**学位论文：**
```
[序号] 作者.题名[D].保存地:保存单位,年份.
```

**专利文献：**
```
[序号] 专利所有者.专利题名[P].专利国别:专利号,发布日期.
```

**技术标准：**
```
[序号] 标准代号,标准名称[S].出版地:出版者,出版年.
```

**报纸文章：**
```
[序号] 作者.题名[N].报纸名,出版日期(版次).
```

**报告：**
```
[序号] 作者.文献题名[R].报告地:报告会主办单位,年份.
```

**电子文献：**
```
[序号] 作者.电子文献题名[文献类型/载体类型].文献网址或出处,发表或更新日期/引用日期.
```

### 10.3 排版要求

- 序号左顶格，用数字加方括号：`[1]`、`[2]`……
- 每条文献末尾以 `.` 结束。
- 外文文献不翻译成中文。
- "参考文献"四字按一级标题编排，内容采用五号字。

---

## 11 附录规范

- 序号采用"附录 1""附录 2"等。
- 附录标题按一级标题编排，各占一行。
- 每个附录一般另起一页，多个较短附录可接排。

---

## 12 打印要求

- A4（210×297mm）白纸，单面印刷（页码过多可双面）。
- 热胶装订，不使用金属钉。

---

## 13 其他注意事项

- 名词术语采用国标规定或行业通用术语，全文统一。
- 英文缩写首次出现时括注全称。
- 量和单位采用国标 GB3100~GB3102（国际单位制 SI）。
- 使用他人表格须注明出处。
- 产品说明书、各类标准（工程设计类除外）、报纸文章、未公开发表的研究报告等不宜作为参考文献。
