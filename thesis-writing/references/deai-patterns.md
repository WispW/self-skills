# 中文学术论文"去 AI 味"模式库

> 综合 [Wikipedia:Signs of AI writing](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing)、
> [Tropes.fyi](https://tropes.fyi/directory) 33 种 AI 写作模式、
> 以及中文本科论文写作场景定制。
>
> **核心原理**：LLM 使用统计算法预测最可能的下一个词，结果倾向于"最通用、最安全"的表达。
> 去 AI 味的本质是：**把统计上最常见的表达，替换为具体的、有个性的、符合真实写作习惯的表达。**

---

## 第一部分：AI 写作特征模式

### A. 内容层面

| # | 模式名称 | 典型表现 | 改写方法 |
|---|---------|----------|---------|
| 1 | **意义夸大** | "具有重要的理论意义和实践价值""至关重要""发挥着关键作用" | 删除空泛评价，改为说明具体贡献了什么 |
| 2 | **肤浅分析** | 在事实后附加"从而实现了…""进而提升了…""有效地促进了…"等分词短语 | 要么给出具体数据支撑，要么删掉这句空话 |
| 3 | **挑战与展望八股** | "尽管…取得了显著进展，但仍面临…挑战""具有广阔的应用前景" | 说明**具体**面临什么挑战，给出**具体**的未来方向 |
| 4 | **模糊归因** | "相关研究表明""学者们普遍认为""有研究指出" | 必须具名引用：张三[1]指出…；如找不到出处则删除 |
| 5 | **叙事完美偏差** | 所有实验/论证都"完美"地支持结论，无失败、无权衡、无局限 | 补充真实的负面结果、对比分析和局限性讨论 |
| 6 | **通用积极结论** | "本文方法取得了良好的效果""实验结果验证了本文方法的有效性" | 改为具体结论：准确率达 92.3%，比基线高 4.1 个百分点 |

### B. 词汇与语法

| # | 模式名称 | 典型表现 | 改写方法 |
|---|---------|----------|---------|
| 7 | **AI 高频词汇** | 赋能、深度、全面、有效、显著、至关重要、不可或缺、核心、关键、助力、旨在 | 删除或替换为具体描述。"全面提升了性能" → "将响应时间从 200ms 降至 80ms" |
| 8 | **连接词滥用** | 每段开头使用"此外""然而""因此""同时""值得注意的是""需要指出的是" | 删掉一半连接词；改用逻辑本身（而非连接词）驱动段落衔接 |
| 9 | **否定式排比** | "不仅…而且…""不是…而是…""不只是…更是…" | 改为直接陈述。"不仅提高了效率，而且降低了成本" → "效率提高 15%，成本下降 8%" |
| 10 | **三段式法则** | "首先…其次…最后…""从…到…再到…" | 打乱为两项或四项；或合并为一句概括 |
| 11 | **规避简单句式** | 回避"是""有"，改用"作为""充当""发挥着…的作用""扮演着…的角色" | 恢复简单句："该模块是系统的核心" 而非"该模块充当着系统核心的角色" |
| 12 | **四字堆砌** | "行之有效、切实可行、深入分析、全面系统、科学合理" 连续出现 | 保留最准确的一个，删除其余 |

### C. 句式与段落结构

| # | 模式名称 | 典型表现 | 改写方法 |
|---|---------|----------|---------|
| 13 | **句式过度整齐** | 连续 3 个以上句子长度相近（±5 字以内）、结构相似 | 主动打乱：长句接短句，复杂句接简单句 |
| 14 | **段落过于工整** | 每段有明确的 topic sentence + 展开 + 总结句，如同模板填写 | 允许部分段落没有总结句；让论述自然收束 |
| 15 | **列表伪装散文** | "第一，…第二，…第三，…" 或隐性编号的排比段落 | 删除编号词，改写为有逻辑递进关系的连贯段落 |
| 16 | **同义词循环** | 对同一概念反复使用不同名称（"该方法""本算法""该方案""此技术"交替使用） | 统一术语；允许合理重复，核心术语全文保持一致 |
| 17 | **分段总结赘余** | 每个小节末尾重复该节内容（"综上，本节介绍了…"） | 只在章末做一次简要小结，段落级不需要总结 |

### D. 中文学术论文特有模式

| # | 模式名称 | AI 写法（改写前） | 人类写法（改写后） |
|---|---------|-------------------|-------------------|
| 18 | **"随着…发展"模板开头** | "随着人工智能技术的快速发展，深度学习在各领域得到了广泛应用。" | "深度学习已在图像分类、语音识别等任务上超越传统方法。" |
| 19 | **"应用前景"模板结尾** | "本文方法在实际应用中具有广阔的应用前景和重要的参考价值。" | "本文方法适用于中小规模数据集的分类任务，在大规模场景下的表现有待验证。" |
| 20 | **"基于 XX 的 XX"过度使用** | "提出了一种基于改进卷积神经网络的基于注意力机制的图像分类方法" | "提出了一种融合注意力机制的图像分类方法，以改进的 CNN 为骨干网络" |
| 21 | **介词结构堆砌** | "基于…，面向…，围绕…，针对…" 在同一段落大量出现 | 控制每段最多 1~2 个介词结构，其余改为主谓宾直接句式 |
| 22 | **"通过…实现了…"公式** | "通过引入注意力机制，实现了对关键特征的有效提取" | "注意力机制使模型在关键区域的特征提取精度提高了 12%" |
| 23 | **贡献声明列表化** | "本文的主要贡献如下：（1）提出了…（2）设计了…（3）实验验证了…" | 保留列表格式（这在学术论文中是可接受的），但确保每条贡献具体而非空泛 |
| 24 | **被动/无主语泛滥** | "可以看出…""由此可见…""不难发现…"连续出现 | 减少使用频率（全文不超过 3~5 次），改为直接陈述事实 |

### E. 语气与交流痕迹

| # | 模式名称 | 典型表现 | 改写方法 |
|---|---------|----------|---------|
| 25 | **协作痕迹残留** | "希望对您有所帮助""如果您需要，我可以…""以下是一个模板" | 全部删除，论文中不应出现对话式语言 |
| 26 | **知识截止免责** | "截至目前""根据现有资料""虽然具体细节有限" | 删除免责声明，改为具体的文献引用和时间标注 |
| 27 | **过度限定** | "可能在一定程度上对结果产生某种影响" | "该因素影响了结果"（需要限定时使用一个限定词即可） |
| 28 | **谄媚/讨好语气** | "这是一个非常好的研究方向""您的分析非常到位" | 删除，论文不包含对读者的评价 |

## 第二部分：中文 AI 高频词汇表

以下词汇在 AI 生成的中文文本中出现频率显著高于人类写作。**单独出现不构成证据**，但在同一段落密集出现（3 个以上）是强信号。

### 形容词/副词类
| AI 偏好 | 人类替代 |
|---------|---------|
| 显著 | 明显 / 具体数据 |
| 全面 | → 说明覆盖了哪些方面 |
| 有效 | → 给出效果数据 |
| 至关重要 | 重要 / 必要 |
| 不可或缺 | 必需 |
| 深入 | → 可删除 |
| 广泛 | → 列举具体范围 |
| 高效 | → 给出效率数据 |

### 动词类
| AI 偏好 | 人类替代 |
|---------|---------|
| 赋能 | 支持 / 帮助 |
| 助力 | 促进 / 帮助 |
| 旨在 | 目的是 / 为了 |
| 探讨 | 分析 / 讨论 |
| 彰显 | 体现 / 表明 |
| 聚焦 | 关注 / 研究 |

### 连接词/过渡语类
| AI 偏好 | 处理方式 |
|---------|---------|
| 值得注意的是 | 通常可直接删除 |
| 需要指出的是 | 通常可直接删除 |
| 此外 | 控制使用频率（全文 ≤3 次）|
| 同时 | 考虑删除或改为"另外"等 |
| 综上所述 | 仅在章节结论处使用 1 次 |
| 总而言之 | 全文最多 1 次 |

### 名词/短语类
| AI 偏好 | 人类替代 |
|---------|---------|
| 应用前景 | → 说明具体适用场景 |
| 理论意义 | → 说明具体理论贡献 |
| 实践价值 | → 说明具体实用场景 |
| 研究范式 | 研究方法 / 研究路径 |
| 技术路线 | → 可保留，但不堆砌 |
| 创新点 | → 可保留，但内容须具体 |

## 第三部分：快速检查清单

在交付去 AI 味文本前，逐项确认：

### 词汇检查
- [ ] 同一段落中"AI 高频词汇"出现 ≤2 个？
- [ ] "此外""然而""因此""同时"等连接词未连续用于相邻段落开头？
- [ ] 无"值得注意的是""需要指出的是"等AI过渡语？
- [ ] "显著""有效""全面"等形容词已替换为具体数据或删除？

### 句式检查
- [ ] 连续 3 个句子长度不相同（差异 ≥10 字）？
- [ ] 无"不仅…而且…""不是…而是…"的否定式排比？（或全文 ≤1 处）
- [ ] 无"首先…其次…最后…"三段式？（或已改为非对称结构）
- [ ] 无"通过…实现了…"的公式化句式？（或全文 ≤2 处）

### 段落检查
- [ ] 每段不以"总之""综上"结尾？（仅章末小结可用）
- [ ] 段落间长度有变化（非全部 150~200 字的均匀段落）？
- [ ] 不以"随着…的发展"开头？

### 内容检查
- [ ] 无空泛的意义声明（"具有重要意义"）？
- [ ] 结论包含具体数据/成果，非泛泛而谈？
- [ ] 存在至少一处真实的局限性讨论？
- [ ] 无协作痕迹（"希望对您有帮助"等）？
- [ ] 无知识截止免责声明？

## 第四部分：处理流程

### 步骤 1：通读识别

通读全文（或用户指定的段落），标记所有匹配上述模式的实例。用 `[DA-#]` 标注（DA = De-AI，# 为模式编号）。

### 步骤 2：分类统计

统计各模式出现次数，按严重程度排序：
- **高频**（≥5 次）→ 全文性问题，需要系统改写
- **中频**（2~4 次）→ 局部处理
- **低频**（1 次）→ 可能是巧合，但仍建议修改

### 步骤 3：逐项改写

按模式编号，从内容层面到表达层面依次改写：

1. **先改内容**：补充缺失的具体数据、真实局限性、批判性分析（模式 1~6）
2. **再改词汇**：替换 AI 高频词为具体描述或更朴素的表达（模式 7~12）
3. **然后调结构**：打乱过于整齐的句式和段式（模式 13~17）
4. **最后改套话**：处理中文学术论文特有的模板化表达（模式 18~24）
5. **清理痕迹**：删除所有对话/交互残留（模式 25~28）

### 步骤 4：整体检查

改写完成后：
1. 大声朗读（或在脑中默读）修改后的段落，听起来是否像**一个真实的本科生在写论文**
2. 检查是否改变了原文的技术含义（不可改变核心观点）
3. **不可过度改写**：目标是自然，不是故意写得差。保留清晰的逻辑和准确的术语
4. 运行"快速检查清单"确认

### 步骤 5：变更说明

输出改写后的文本，并附简要的变更列表：
```
已处理的 AI 特征：
- [DA-7] 替换了 3 处"显著"为具体数据
- [DA-8] 删除了 2 处段首"此外"
- [DA-18] 改写了开头"随着…发展"模板句
```

## 第五部分：质量评分

对去 AI 味处理后的文本打分（满分 50 分）：

| 维度 | 评估标准 | 得分 |
|------|----------|------|
| **具体性** | 是否用具体数据/事实替代了空泛描述？<br>10 分：全用具体描述；1 分：仍是空话 | /10 |
| **节奏感** | 句子长短是否有变化？段落是否有疏密？<br>10 分：长短交错自然；1 分：机械均匀 | /10 |
| **朴素度** | 是否回避了 AI 高频词汇和华丽套话？<br>10 分：朴素直接；1 分：堆砌辞藻 | /10 |
| **真实感** | 读起来像真人写的吗？有血有肉还是冷冰冰？<br>10 分：自然真实；1 分：机器味浓 | /10 |
| **准确性** | 改写是否保持了原文的技术含义和逻辑？<br>10 分：无失真；1 分：严重偏离 | /10 |
| **总分** |  | **/50** |

**评价标准**：
- **45-50 分**：优秀，AI 痕迹已基本消除
- **35-44 分**：良好，仍有改进空间
- **25-34 分**：一般，需要重点修改
- **低于 25 分**：需要大幅重写

## 第六部分：完整示例

### 示例 1：绪论段落

**改写前（AI 味浓）：**

> 随着人工智能技术的快速发展，深度学习在计算机视觉领域取得了显著的成果。卷积神经网络作为深度学习的核心技术之一，在图像分类、目标检测和语义分割等任务中发挥着至关重要的作用。然而，传统的卷积神经网络在处理复杂场景时仍面临诸多挑战。因此，如何有效地提升模型的特征提取能力，成为了当前研究的热点问题。此外，注意力机制的引入为解决上述问题提供了新的思路。综上所述，研究基于注意力机制的图像分类方法具有重要的理论意义和实践价值。

**问题标注**：
- [DA-18] "随着…快速发展"模板开头
- [DA-7] "显著的成果""至关重要""有效地"
- [DA-11] "作为…之一""发挥着…的作用"
- [DA-3] "仍面临诸多挑战"
- [DA-8] "然而""因此""此外""综上所述" 连续堆砌
- [DA-1] "重要的理论意义和实践价值"

**改写后（去 AI 味）：**

> 卷积神经网络（CNN）在 ImageNet 分类任务上将 Top-5 错误率从 26% 降至 3.6%[1]，但在遮挡、小目标等复杂场景下准确率仍会下降 10~15 个百分点[2]。2017 年提出的自注意力机制为此提供了一条路径——通过动态调整不同空间位置的权重，模型可更集中地关注目标区域[3]。本文在 ResNet-50 骨干网络上嵌入通道-空间双注意力模块，目标是在不显著增加计算量的前提下提高复杂场景的分类精度。

### 示例 2：实验结论段落

**改写前（AI 味浓）：**

> 实验结果表明，本文提出的方法在多个评价指标上均优于现有方法，充分验证了该方法的有效性和优越性。通过引入注意力机制，有效地提升了模型的特征提取能力，从而实现了对复杂场景的精准识别。同时，消融实验进一步证明了各模块的贡献。总而言之，本文方法在图像分类任务中取得了良好的效果，具有广阔的应用前景。

**改写后（去 AI 味）：**

> 在 CIFAR-100 测试集上，本文方法的 Top-1 准确率为 82.7%，比 ResNet-50 基线高 3.2 个百分点，比 SE-Net 高 1.1 个百分点（表 5-1）。消融实验显示，通道注意力模块贡献了约 2.1 个百分点的提升，空间注意力模块贡献了约 1.4 个百分点，两者存在 0.3 个百分点的重叠效应。需要说明的是，本文实验仅在 224×224 分辨率图像上进行，对于更高分辨率场景的效果尚未验证。
